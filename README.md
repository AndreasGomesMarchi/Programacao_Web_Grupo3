<p>
Execute esses comandos para executar o programa.</p>
<p>1 npx drizzle-kit generate: Este comando é o responsável por ler o seu arquivo schema.ts e criar os arquivos de migração (scripts SQL).</p>
<p>2 npx drizzle-kit push: Este comando pega as alterações e as aplica diretamente no seu arquivo de banco de dados (database.sqlite).</p>
<p>3 node --loader ts-node/esm src/index.ts: Este é o comando que inicia o seu programa para que você possa interagir com o menu.</p>
